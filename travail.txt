💻 CHANGER DE PC — Comment continuer à travailler
Tout votre code est sur GitHub . Changer de PC est donc très simple.

Sur le nouveau PC, installez ces outils :
1. Git
https://git-scm.com/download/win
→ Télécharger et installer avec les options par défaut
2. Node.js 20 LTS
https://nodejs.org
→ Télécharger la version LTS (20.x)
→ Installer avec les options par défaut
3. pnpm
PowerShellnpm install -g pnpm
4. Docker Desktop
https://www.docker.com/products/docker-desktop/
→ Même installation que ce qu'on a fait aujourd'hui

Récupérer le projet depuis GitHub
PowerShell# 1. Cloner le dépôt
git clone https://github.com/VOTRE_USERNAME/safechild-africa.git
cd safechild-africa

# 2. Installer toutes les dépendances
#    (pnpm lit le pnpm-workspace.yaml et installe tout)
pnpm install

# 3. Recréer le fichier .env
#    (il n'est PAS sur GitHub car dans .gitignore)
#    → Copiez votre .env depuis l'ancien PC
#    OU recréez-le manuellement
copy .env.example .env
#    Puis remplissez les valeurs

# 4. Démarrer Docker
pnpm docker:up

# 5. Démarrer le projet
pnpm dev
